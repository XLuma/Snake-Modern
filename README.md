# Snake-Modern
Reimplementation of Snake Muxxu by Motion-Twin.

# Setup
Install nvm on your machine.

1. Clone
2. `nvm use`
3. `cd server && cp .env-dist .env`
4. Setup the env with your proper db url
5. `npm install`
6. `npm run dev`
